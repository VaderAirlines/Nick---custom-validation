Bij het starten van de pagina wordt de default option value getoond. Deze komt zogezegd van de styleguide.

Druk je op de knop 'show option value' wordt de huidige option value ge-alert.

Via de knop 'set value' (en een ingevulde waarde in het tekstvak) kan je de default option value overschrijven.

Druk je dan terug op 'show option value' kan je het resultaat zien.

Wil je de custom value terug verwijderen kan je dat doen door op de knop 'remove custom option value' te drukken.

Je kan dan alweer de waarde controleren door de knop 'show option value'
